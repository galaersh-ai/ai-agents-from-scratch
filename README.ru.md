# AI Agents From Scratch

Изучите создание AI-агентов локально без фреймворков. Понимайте, что происходит «под капотом», прежде чем использовать производственные решения.

![Обзор архитектуры агента](diagrams/agent-architecture.png)


## Назначение

Этот репозиторий учит создавать AI-агентов с нуля, используя **локальные LLM** и **node-llama-cpp**. Работая с примерами, Вы поймёте:

- Как LLM работают на фундаментальном уровне
- Что такое агенты на самом деле (LLM + инструменты + паттерны)
- Как работают различные архитектуры агентов
- Почему фреймворки принимают те или иные решения

> Python-версия этого туториала доступна здесь:
> https://github.com/pguso/agents-from-scratch

**Философия**: Учитесь через практику. Понимайте глубоко, затем мудро используйте фреймворки.

## Сопутствующий сайт

У этого репозитория есть **сопутствующий сайт**:

**https://agentsfromscratch.com**

Сайт **не заменяет** репозиторий, а является **концепуальным компаньоном**, который:

- Объясняет *зачем* существует каждый пример
- Визуализирует путь обучения от сырых вызовов LLM до полноценных агентов
- Разделяет **код**, **объяснения** и **ключевые концепции**
- Помогает понять архитектуры агентов до использования фреймворков

**Рекомендуемый рабочий процесс:**
- Используйте **GitHub** для запуска, модификации и изучения кода

> Считайте сайт *картой*, а этот репозиторий — *местностью*.

## Основы агентов — от LLM до ReAct

### Предварительные требования
- Node.js 18+
- Минимум 8GB RAM (рекомендуется 16GB)
- Скачайте модели и поместите в папку `./models/`, подробности в [DOWNLOAD.md](DOWNLOAD.md)

### Установка
```bash
npm install
```

### Запуск примеров
```bash
node intro/intro.js
node simple-agent/simple-agent.js
node react-agent/react-agent.js
```

## Ключевые концепции

### Что такое AI-агент?

```
AI Agent = LLM + Системный промпт + Инструменты + Память + Паттерн рассуждений
           ─┬─   ──────┬──────   ──┬──   ──┬───   ────────┬────────
            │          │           │       │              │
         Мозг      Идентичность  Руки   Состояние     Стратегия
```

### Эволюция возможностей

```
1. intro          → Базовое использование LLM
2. translation    → Специализированное поведение (системные промпты)
3. think          → Способность к рассуждениям
4. batch          → Параллельная обработка
5. coding         → Стриминг и контроль
6. simple-agent   → Использование инструментов (вызов функций)
7. memory-agent   → Постоянное состояние
8. react-agent    → Стратегические рассуждения + использование инструментов
```

### Архитектурные паттерны

**Простой агент (шаги 1-5)**
```
Пользователь → LLM → Ответ
```

**Агент с инструментами (шаг 6)**
```
Пользователь → LLM ⟷ Инструменты → Ответ
```

**Агент с памятью (шаг 7)**
```
Пользователь → LLM ⟷ Инструменты → Ответ
       ↕
     Память
```

**ReAct-агент (шаг 8)**
```
Пользователь → LLM → Думать → Действовать → Наблюдать
       ↑      ↓      ↓      ↓
       └──────┴──────┴──────┘
           Итерировать до решения
```

- Историю разговора
- Состояние контекста

Пример использования в `simple-agent/simple-agent.js`
