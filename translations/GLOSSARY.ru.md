## Правила перевода терминов

1. **Устоявшийся перевод есть** → используем его (model → модель, agent → агент)
2. **Нет устоявшегося** → транслитерация (token → токен, embedding → эмбеддинг)
3. **Аббrevиатуры** → без перевода (LLM, GPU, GGUF, API)
4. **Названия библиотек** → без перевода (node-llama-cpp, PyTorch)
5. **Глаголы** → адаптируем (to train → обучать, to infer → выполнять инференс)
6. **Паттерны агентов** → без перевода если международные (ReAct, AoT)
7. **Параметры моделей** → temperature → температура, context window → контекстное окно
